## Getting Started

### Option 1: Direct Opening
Simply open the `index.html` file in your web browser. No server needed!

### Option 2: Live Server (Recommended)
If you're using VS Code:
1. Install the "Live Server" extension
2. Right-click on `index.html`
3. Select "Open with Live Server"

### Option 3: Local Server
Using Python:
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --lavender-light: #E6E6FA;
    --lavender-dark: #9370DB;
    --purple-light: #BA55D3;
    /* Add your custom colors */
}
```

### Adding New Sections
1. Add HTML structure in `index.html`
2. Style it in `styles.css`
3. Add animations in `script.js` if needed

### Updating Content
Simply edit the content in `index.html` - all sections are clearly marked with comments.

## Deployment

### GitHub Pages
1. Create a GitHub repository
2. Upload all files
3. Go to Settings > Pages
4. Select main branch
5. Your site will be live at `username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Your site is live instantly!

### Vercel
1. Install Vercel CLI
2. Run `vercel` in the project directory
3. Follow the prompts
